
Project to recognise and translate sign language in real time

# Getting Started

```
$ git clone https://github.com/Anon9559/sign_language
```

Make a virtual environment

```
$ cd sign_language
$ python3 -m venv env
$ . env/bin/activate
```

Install the requirements

```
(env) $ pip install -r requirements.txt
```
